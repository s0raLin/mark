# 使用文档

## 启动应用

### Electron 版本

```bash
pnpm dev:electron
```

### Tauri 版本

```bash
cd src-tauri
cargo tauri dev
```

## 基本使用

### 创建文件

- 点击侧边栏上的新建文件按钮
- 输入文件名
- 如果没有扩展名，系统会自动补 `.md`

### 创建文件夹

- 点击侧边栏上的新建文件夹按钮
- 输入文件夹名

### 打开文件

- 在侧边栏点击目标文件
- 编辑器会自动加载内容

### 搜索文件

- 点击搜索按钮
- 或使用快捷键 `Ctrl/Cmd + K`
- 支持按文件名和文件内容搜索

### 保存

- 编辑内容后会自动保存
- 也可以使用快捷键 `Ctrl/Cmd + S`

### 切换视图

顶部工具栏支持：

- 分栏模式
- 仅编辑器
- 仅预览

### 设置主题

打开设置后可修改：

- 编辑器主题
- 预览主题
- 强调色
- 背景图
- 字体
- 自动保存

## 错误提示说明

应用内的错误提示遵循以下原则：

- 用户优先看到可理解的提示语
- 技术错误只在开发环境显示详细信息
- 短时间重复错误会合并，避免连续刷屏

常见提示示例：

- `初始化数据加载失败`
  应用会继续以默认状态启动
- `自动保存失败`
  当前改动还在编辑器里，不代表内容已丢失
- `操作没有完成`
  一般表示本地后端或运行时通信失败

## 数据存储位置

### Tauri / Rust

用户数据默认写入系统配置目录下的 `notemark` 目录。

### Electron / Go

Electron 旧链路同样使用系统配置目录下的本地文件。

主要包括：

- `userdata.json`
- `files/`
- `uploads/`

## 当前运行模式如何判断

如果你是通过 `pnpm dev:electron` 启动，一般运行在 Electron 模式。

如果你是通过 `cargo tauri dev` 启动，一般运行在 Tauri 模式。

前端会自动适配，不需要手动切换。
